Python Clipboard

Mar 6, 2022

The script allows users to save multiple keywords and access them on-demand. It uses pyperclip to paste the words stored in sys.argv into a file called mcb.

Listing the keywords pastes the content of file mcb into the clipboard.

['third', 'fourth', 'second', 'first']

Code Breakdown

sys.argv is a list in Python that contains command-line arguments passed to the script. The sys.argv list always has the script file name at index 0. When the user calls the script with save keyword, an if statement checks if sys.argv has three elements. The script name is at index 0, save at index 1, and the keyword at index 2. In that case, pyperclip pastes the keyword at index 2 to the file mcb.

The delete command works similarly to save but removes the keyword from mcb instead.

If sys.argv receives only a command and no keyword, it checks if it's list or clear. The list command copies all keywords from mcb to the clipboard as a string.

The clear command loops over all keywords in the file and deletes them one by one.

Finally, if the user wants a saved keyword pasted into the clipboard, they can type:

py mcb.pyw keyword
