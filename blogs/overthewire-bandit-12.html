OverTheWire Bandit Level-12 Writeup

Sep 14, 2021

OverTheWire Bandit is a Linux CTF that's challenging but fun. I chose to write about this specific bandit because of how challenging it was.

Goal of the Challenge

The password for the next level is stored in the file data.txt, which is a hexdump of a file that has been repeatedly compressed. For this level, it may be useful to create a directory under /tmp in which you can work using mkdir. Then copy the datafile using cp, and rename it using mv (read the manpages!).

We are given a list of Linux commands that might be useful: grep, sort, uniq, strings, base64, tr, tar, gzip, bzip2, xxd, mkdir, cp, mv, file. More details on Hex dump.

SSH into the Host

ssh bandit.labs.overthewire.org -p 2220 -l bandit12

Password: 5Te8Y4drgCRfCx8ugdwuEX8KFC6k2EUu

Finding and Copying data.txt

bandit12@bandit:~$ ls
data.txt
bandit12@bandit:~$ mkdir /tmp/myname123
bandit12@bandit:~$ cp data.txt /tmp/myname123
bandit12@bandit:~$ cd /tmp/myname123
bandit12@bandit:/tmp/myname123$ ls
data.txt

Reversing the Hex Dump

bandit12@bandit:/tmp/myname123$ xxd -r data.txt data
bandit12@bandit:/tmp/myname123$ file data
data: gzip compressed data, was "data2.bin", last modified...

Decompression Steps

bandit12@bandit:/tmp/myname123$ mv data data.gz
bandit12@bandit:/tmp/myname123$ gzip -d data.gz

The process repeats with different compression types: bzip2, gzip, tar, until we finally extract the password.

bandit12@bandit:/tmp/myname123$ file data8
data8: ASCII text
bandit12@bandit:/tmp/myname123$ cat data8
The password is 8ZjyCRiBWFYkneahHwxCv3wb2a1ORpYL

Conclusion

The password for the next level, bandit13, is 8ZjyCRiBWFYkneahHwxCv3wb2a1ORpYL.
